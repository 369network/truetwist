04 — Iconography

Icon System

A duo-tone icon style combining outlined structure with selective filled accents. Rounded corners and consistent stroke weight create a friendly, modern feel aligned with the brand.

Option A
Outline
Clean, lightweight. Best for dense UI and navigation.
Recommended
Duo-Tone
Outline + soft fill accent. Adds depth and brand color without heaviness.
Option C
Filled
Bold, high contrast. Best for CTAs, feature highlights, and emphasis.
