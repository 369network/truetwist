06 — Brand Application
Template Brand Guidelines
Rules for applying user brand assets consistently across all auto-generated templates.
RuleSpecification
Logo PlacementAlways in a corner. Default: bottom-right. Minimum 5% margin from edges. Max 8% of shortest edge. Semi-transparent (40% opacity) when overlapping content.
Primary Color UsageBackground fills, CTA buttons, headline accents. Must pass WCAG AA contrast (4.5:1) against text. Auto-lightens or darkens for dark/light themes.
Secondary Color UsageAccents only: CTA highlights, decorative elements, alert badges. Never as background for large areas. Max 20% of visual area.
Font ApplicationHeading font for headlines and CTAs. Body font for descriptions. If custom fonts unavailable, fall back to Outfit → system sans-serif.
Image TreatmentAI-generated backgrounds receive brand color overlay (10-20%). Product images: no overlay, clean cutout with shadow. Person images: subtle vignette.
Dark vs LightTemplates auto-detect brand color luminance. Light brands → dark text. Dark brands → light text. Override available in API.
AccessibilityAll text must meet WCAG AA (4.5:1 normal, 3:1 large). Auto-adds text shadow/backdrop when contrast is insufficient over images.
